# Beasts Registry (神兽系统)

This folder contains the classified archive of mythological beasts within the AI Myth Universe.

Each entity is documented in an "official archive format" as if it were part of a real cosmological database.

---

## Classification System

### Threat Levels
- L0: Harmless / Symbolic
- L1: Local anomaly
- L2: Regional disruption
- L3: City-level threat
- L4: Continental threat
- L5: Reality-class anomaly

### Origin Types
- Natural evolution
- Divine creation
- Abyss corruption
- Human myth formation
- Unknown origin

---

## Standard Entry Format

Each beast file follows:

```
ID: BX-000
Name: 
Type: 
Threat Level: 
Origin: 
Habitat: 
Abilities: 
Description: 
```

---

## Purpose

To build a scalable mythological bestiary that can be used for:
- Worldbuilding
- AI image generation prompts
- Game design
- Story scripts
