# AI Myth Universe (AI 神话宇宙)

This repository is the foundation of a generated mythological world built by AI.

## Concept

A continuously expanding fictional universe that merges multiple mythologies into a single coherent system:

- Chinese mythology (山海经, 道教神话)
- Norse mythology
- Egyptian mythology
- Greek-inspired divine systems
- Original AI-generated deities and creatures

The universe is treated as an **archival system**, not a storybook.
Each entity is documented as if it were real.

## Structure

### 1. Entities
- Gods
- Beasts
- Spirits
- Ancient civilizations

### 2. Regions
- Mythic continents
- Divine realms
- Forbidden zones

### 3. Historical timeline
- Creation events
- Divine wars
- Collapse eras

### 4. Documentation format
Each entry follows a strict archive style:

```
ID: MX-001
Name: Example Entity
Type: Divine Beast
Threat Level: Unknown
Description: ...
```

## Goal

To build a scalable AI-generated mythology database that can later be used for:
- Web storytelling site
- Short video scripts
- Game worldbuilding
- Visual AI generation

Early prototype stage.
